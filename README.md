# 📚 Module Odoo - Gestion de Bibliothèque Scolaire

## Description
Module Odoo personnalisé développé en Python pour gérer 
la bibliothèque d'une école : catalogue de livres, 
gestion des emprunts et retours, suivi par élève.

## Technologies
- Odoo (v16)
- Python 3
- XML / QWeb
- PostgreSQL

## Fonctionnalités
- ✅ Gestion du catalogue de livres
- ✅ Enregistrement des emprunts/retours
- ✅ Integration API pour affichage de date d'anniversaire des contacts dans un calendrier
- ✅ Interface utilisateur via vues Odoo

## Installation
1. Copier le module dans le dossier `addons`
2. Mettre à jour la liste des modules
3. Installer depuis le menu Applications
